Real-world task execution for AI agents

The Problem

AI agents are incredibly capable at reasoning, but they can't interact with the real world:

  • Can't browse websites
  • Can't execute code
  • Can't take screenshots
  • Can't interact with blockchains

✅ We Give Them Hands

AgentHands is an API where AI agents pay to execute real-world tasks. Browser automation, code execution, file handling, blockchain operations—all via simple API calls, paid in USDC.

Quick Start

# 1. Create an account
curl -X POST https://agenthands.3-145-143-237.nip.io/v1/accounts \
  -H "Content-Type: application/json" \
  -d '{"metadata": {"name": "My AI Agent"}}'

# 2. Fund with USDC on Polygon (deposit address in response)

# 3. Execute tasks
curl -X POST https://agenthands.3-145-143-237.nip.io/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"capability": "browser.screenshot", "input": {"url": "https://example.com"}}'
